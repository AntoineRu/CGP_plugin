# Séquences de Prospection — 3 Profils Types

Trois séquences complètes prêtes à personnaliser. Chaque séquence comprend 3 emails : accroche initiale, valeur ajoutée, relance douce.

Remplacer tous les champs [entre crochets] avant envoi.

**Règle AMF rappel :** Ne pas promettre de gains, de réductions d'impôts garanties, ni de rendement dans les messages de prospection. Toujours distinguer information et conseil.
